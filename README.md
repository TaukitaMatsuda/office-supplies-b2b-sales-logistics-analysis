**Ключевая находка**  
Выручка стабильно растёт (+сезонность с пиком осенью), 97%+ выручки приносят корпоративные клиенты (B2B), при этом после 2018 года почти не приходят новые клиенты → основной рост за счёт удержания старых.



**Продуктовые / бизнес-метрики (ключевые)**

| Метрика                              | Значение                  | Комментарий                                       |
|--------------------------------------|---------------------------|---------------------------------------------------|
| Доля выручки от B2B (Corporate)      | ~97%                      | Почти весь бизнес зависит от корпоративных клиентов |
| Средний чек корпоративных клиентов   | Значительно выше B2C      | B2B — основной драйвер прибыли                     |
| Новые корпоративные клиенты после 04.2018 | Почти 0                 | Критическая проблема притока                      |
| Топ-категория по выручке             | Chairs / Office Furniture | Кресла — явный лидер продаж                       |
| Доля своевременных доставок (First Class) | 99.87%                | Но Second Class — только 79%                      |
| Лучший штат по выручке и заказам     | California                | Лидер по всем метрикам                            |

**Основные выводы и продуктовые рекомендации**

1. **Критическая зависимость от старых B2B-клиентов**  
   После апреля 2018 года приток новых корпоративных клиентов почти остановился → основной рост за счёт повторных покупок существующих.  
   **Рекомендация P0**: Активизировать B2B-маркетинг (LinkedIn, холодные продажи, партнёрства с офисными центрами).

2. **Сильная сезонность и лидерство категории «Кресла»**  
   Пик продаж осенью (начало учебного года + корпоративные закупки).  
   Топ-товары: кресла HON, копиры Canon, 3D-принтеры.  
   **Рекомендация P1**: Расширить ассортимент в категории executive chairs и large-format офисной техники → это может увеличить средний чек на 15–25%.

3. **Проблемы с логистикой (Second Class доставка)**  
   ~21% заказов Second Class приходят с опозданием → может быть одной из причин оттока новых клиентов.  
   **Рекомендация P1**: Пересмотреть партнёров по доставке или перевести больше заказов на First Class / Same Day в ключевых штатах.

4. **География — явный лидер California**  
   По количеству заказов, выручке и объёму — Калифорния на первом месте.  
   **Рекомендация P0**: Открывать первый оффлайн-магазин/шоурум именно в Калифорнии (Лос-Анджелес / Сан-Франциско).  
   Вторые приоритеты: Нью-Йорк → Техас.
